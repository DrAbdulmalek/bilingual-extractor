# BilingualExtractor - CHANGELOG

## [Unreleased]

### Added
- Initial TermExtractor module with pattern-based extraction
- TextPreprocessor with Arabic normalization and OCR error correction
- PatternExtractor with bilingual pair detection
- MorphologicalAnalyzer with medical domain categorization
- StatisticalFilter with confidence scoring and deduplication
- CSV, TSV, JSON, Excel export functionality
- Comprehensive medical terminology dictionaries
- Dialectal variant mapping (Egyptian, Levantine, Gulf)
- Historical terminology evolution tracking
- CLI interface for batch processing
- Interactive Jupyter notebook for Google Colab
- Database schema (PostgreSQL) for corpus management
- Bilingual documentation (Arabic + English)
- PR template for contributions
